## Burger Builder Application w/ ReactJS

### What I used?

- React 16 / Jsx pages
- React Router
- Axios
- Webpack
- Babel
- Vanilla JavaScript
- HTML5
- CSS3 w/ Mobile First approach
- FireBase for Database and Hosting

## How Can I Start?

<pre>npm install</pre>
then
<pre>npm start</pre>
